# 🎵 Music Queue System

A beginner Python project that simulates a music player queue using the Queue data structure (FIFO - First In First Out).

## 🚀 Features

* Add songs to the queue
* Play the next song
* View the current queue
* Handle empty queue situations
* Exit the program

## 🧠 Concepts Used

* Python Lists
* Queue Data Structure
* FIFO (First In First Out)
* Loops
* Conditional Statements
* User Input

## 📚 How Queue Works

Queue follows the FIFO rule:

**First In → First Out**

Example:

Queue:

Song1
Song2
Song3

Play Next Song:

Song1 is played first and removed from the queue.

Remaining Queue:

Song2
Song3

## ▶️ Example Menu

1. Add Song
2. Play Next Song
3. Show Queue
4. Exit

## 🎯 Learning Outcome

This project helps understand:

* Queue operations
* FIFO behavior
* List manipulation in Python
* Building menu-driven programs
